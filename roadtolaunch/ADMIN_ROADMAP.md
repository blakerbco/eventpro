# Admin Control Panel — Full Roadmap

## Phase 1: Launch to 10 Businesses (NOW)
- [x] Admin Dashboard — KPI cards (revenue, users, operations, health)
- [x] User Management — searchable list, detail page, ban/unban, wallet adjust
- [x] Revenue — 30-day daily breakdown, top 10 spenders
- [x] Activity — recent jobs, recent logins
- [x] System — running jobs, cache stats, drip stats, server info
- [x] Ban System — block login, clear session, ban/unban from admin panel

## Phase 2: Week 1-2 (Post-Launch Polish)
- [ ] Audit Log — track all admin actions (ban, wallet adjust, etc.) with timestamps
- [ ] Login History — full login history per user (IP, timestamp, user agent)
- [ ] Promo Code Management — create/edit/deactivate promo codes from admin
- [ ] Refund Tooling — one-click refund for a transaction, auto-credit wallet
- [ ] Export History — download user list, revenue report, job history as CSV

## Phase 3: Month 1 (Security & Operations)
- [ ] Impersonation — "login as user" for debugging (with audit trail)
- [ ] IP Controls — view login IPs, block suspicious IPs, geo-flagging
- [ ] Fraud Signals — flag accounts with suspicious patterns (rapid signups, abuse)
- [ ] Lead Disputes — users can dispute lead quality, admin review queue
- [ ] Feature Flags — toggle features per user or globally (API access, exclusive leads, etc.)

## Phase 4: Scale (10+ Businesses)
- [ ] 2FA for Admin — require TOTP for admin login
- [ ] Team Accounts — multiple users under one organization/billing
- [ ] Cohort Analytics — signup cohorts, retention curves, LTV analysis
- [ ] Risk Scoring — automated fraud scoring per user based on behavior
- [ ] Content Management — editable landing page sections, email templates from admin

## Original Wishlist (10 Sections)
1. Dashboard (KPIs, health metrics)
2. User Management (CRUD, ban, wallet, search)
3. Revenue & Billing (timeline, per-user, refunds)
4. Job Monitoring (live jobs, history, cost analysis)
5. Cache Management (stats, purge, refresh)
6. Support Tickets (admin view, priority queue)
7. Exclusive Leads (sold leads, disputes, analytics)
8. Email & Drip Campaigns (stats, manual triggers)
9. System Health (uptime, errors, API usage)
10. Security (audit log, IP tracking, 2FA)
